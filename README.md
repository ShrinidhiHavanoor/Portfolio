# Shrinidhi Havanoor | Professional Portfolio

A high-performance, mesmerizing single-page portfolio built with React, TypeScript, Tailwind CSS v4, and Framer Motion.

## Features
- **Modern Aesthetic:** Dark mode with Glassmorphism and vibrant gradients.
- **Interactive Elements:** Smooth scroll animations, typing effects, and hover transitions.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop.
- **Project Showcase:** Highlights key backend projects like Divastra and SaaS Contact Form.
- **Tech Stack:** 
  - Frontend: React 19, Vite, Tailwind CSS v4
  - Icons: Lucide React
  - Animations: Framer Motion

## Getting Started

1.  **Clone the repository** (or download the source).
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run in development mode:**
    ```bash
    npm run dev
    ```
4.  **Build for production:**
    ```bash
    npm run build
    ```

## Sections
- **Hero:** Personal introduction with typing animation.
- **About:** Career summary and key statistics.
- **Experience:** Professional journey at Rakuten India.
- **Skills:** Interactive grid of core strengths.
- **Projects:** Card-based showcase of featured work.
- **Education:** Vertical timeline of academic history.
- **Certifications:** Grid of professional achievements.
